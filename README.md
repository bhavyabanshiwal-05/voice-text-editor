# Voice Text Editor Notepad

## 📌 Description
This project is a Python-based voice text editor that allows users to write text using voice commands and translate it into multiple languages.

## 🚀 Features
- Voice typing
- Multilingual translation
- GUI-based text editor
- File handling (open, save)
- Language selection

## 🛠️ Technologies Used
- Python
- Tkinter
- SpeechRecognition
- Deep Translator

## ▶️ How to Run
1. Install required libraries:
   pip install SpeechRecognition deep-translator pyaudio

2. Run the project:
   python voice_editor.py

## 📷 Output
Real-time voice typing and translation system.
